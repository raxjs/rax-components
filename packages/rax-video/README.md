## 描述
视频播放组件

## 安装

```bash
$ npm install rax-video --save
```

## 示例
[在线 Demo](https://jsplayground.taobao.org/raxplayground/0c5fb709-9444-4b83-b01b-944356f1020d)
```jsx
import {createElement, Component, render} from 'rax';
import Video from 'rax-video';
import DriverUniversal from 'driver-universal';

render(<Video style={{ width: 750, height: 400 }} autoPlay src="https://cloud.video.taobao.com/play/u/2780279213/p/1/e/6/t/1/d/ld/36255062.mp4" />, document.body, { driver: DriverUniversal });
```
